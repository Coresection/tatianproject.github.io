A) PHP Files and the HTML Structure - top

In this theme we implemented the responsive design with 3 different layout schemes:

1) Desktop Design: has a width of 1200px (which contains activated at minimum screen width of 1250px).
2. Tablet Design: has a width of 650px (which contains activated at maximum screen width of 1250px)
3. Mobile Design: has a width of 300px (which contains activated at maximum screen width of 700px)

We put our focus in the coding part on the quality and scalability of these three layouts.

In the main CSS file which has the name "style.css" and can be found in the root of the template css/style.css, contains all three design layouts. How exactly these three parts (desktop, tablet and mobile version) can be found in the style.css file, we explain it here.

We created a beautiful html structure for better and fast understanding for the first review.
We even commented the beginning and ending of every main div container.



Here are listed all the PHP files

  1. index.php - The main file and homesite of the website
  2. header.php - Contains the navigation of the page - Just change the navigation here once and its changed on all pages
  3. booking.php - Contains the booking send form of the index page - Here you can change the receiver email address of the form
  4. menu.php - The menu page on which the food menu is listed
  5. about.php - The about page of the Tatian restaurant website
  6. team.php - The team page of the Tatian restaurant website
  7. news.php - The news page of the Tatian restaurant website
  8. contact.php - The contact page on which the map is embedded and contactform and also contact informations of the Tatian restaurant

B) CSS Files and Structure - top

Basically I'm using only one file for the whole website design but two others which are font-awesome icons libraries. So there are total three CSS files in this theme. The main "style.css" is the main file for all style on the theme. This file contains all general styling, such as anchor tag colors, font-sizes, background-color, height, width of div container etc. Keep in mind, that some of these values might be overridden somewhere else in the file, where the responsive design takes action for example for the Tablet design or mobile design.

In the main css "style.css "file, the Responsive Design starts at the following point with the Tablet Design:

	/********** Responsive Design START **********/
	/********** Tablet Design START **********/

	@media screen and (max-width: 1250px) {
		...here is the overridden code for the tablet design...
	}

And at this point, the Mobile Design starts:

	/********** Mobile Design START **********/
	@media screen and (max-width: 700px) {
		...here is the overridden code for the mobile design...
	}

If you would like to edit a specific section of the site, simply find the appropriate label in the CSS file, and then scroll down until you find the appropriate style that needs to be edited.

	#.readmore.one {
		background-color: red; (if you dont like that color, delete this line and create like the following:)
		background-color: blue; (the new color you gave to it)
	}

Here are listed all the CSS files

F) .htaccess File - top

The reason why the files are in PHP, is because we use PHP in many file sections. Thats why we used PHP and rewrite the .php ending with the .htaccess file.

What you need to do is, just upload the whole theme after your custom changing onto your webserver, and change the "htaccess" file to ".htaccess". If files have already a dot in the beginning, they get invisible in the Windows explorer or OSX finder.

After doing this proceder of yours, all pages will show up with .html ending in the browser.
For example: contact.php will be shown as contact.html.
